## 1. Install & configure

<CodeBlocks>
  <CodeBlock title="Python">
    ```bash
    pip install browser-use-sdk
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```bash
    npm install browser-use-sdk
    ```
  </CodeBlock>
</CodeBlocks>

Get a key at [cloud.browser-use.com](https://cloud.browser-use.com), then:

```bash
export BROWSER_USE_API_KEY=your_key
```

## 2. Run your first task

<CodeBlocks>
  <CodeBlock title="Python">
    ```python
    import asyncio
    from browser_use_sdk import AsyncBrowserUse

    async def main():
        client = AsyncBrowserUse()
        result = await client.run("What is the top post on Hacker News right now?")
        print(result.output)

    asyncio.run(main())
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { BrowserUse } from "browser-use-sdk";

    const client = new BrowserUse();
    const result = await client.run("What is the top post on Hacker News right now?");
    console.log(result.output);
    ```
  </CodeBlock>
</CodeBlocks>

## 3. Get structured data

Define a schema and get validated, typed data back.

<CodeBlocks>
  <CodeBlock title="Python">
    ```python
    import asyncio
    from browser_use_sdk import AsyncBrowserUse
    from pydantic import BaseModel

    class Post(BaseModel):
        title: str
        url: str
        points: int

    async def main():
        client = AsyncBrowserUse()
        result = await client.run(
            "Get the top 3 posts from Hacker News",
            output_schema=list[Post],
        )
        for post in result.output:
            print(f"{post.title} ({post.points} pts)")

    asyncio.run(main())
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { BrowserUse } from "browser-use-sdk";
    import { z } from "zod";

    const Post = z.object({
      title: z.string(),
      url: z.string(),
      points: z.number(),
    });

    const client = new BrowserUse();
    const result = await client.run("Get the top 3 posts from Hacker News", {
      schema: z.array(Post),
    });
    result.output.forEach((post) => console.log(`${post.title} (${post.points} pts)`));
    ```
  </CodeBlock>
</CodeBlocks>
