---
title: Profiles
description: "Persistent browser state — cookies, localStorage, saved passwords. Login once, reuse across sessions."
icon: user
---

A profile saves your browser state. Login once, and the agent is already authenticated in every future session.

<CodeGroup>
```python Python
from browser_use_sdk import AsyncBrowserUse

client = AsyncBrowserUse()
session = await client.sessions.create(profile_id="profile_abc123")
result = await client.run("Check my GitHub notifications", session_id=session.id)
print(result.output)
```
```typescript TypeScript
import { BrowserUse } from "browser-use-sdk";

const client = new BrowserUse();
const session = await client.sessions.create({ profileId: "profile_abc123" });
const result = await client.run("Check my GitHub notifications", {
  sessionId: session.id,
});
console.log(result.output);
```
</CodeGroup>

View your profiles at [cloud.browser-use.com/settings](https://cloud.browser-use.com/settings?tab=profiles).
