---
title: Live preview
description: "Embed the agent's live browser view in your app using an iframe."
---

Every session has a `liveUrl` — a real-time view of the agent's browser. Embed it in your app so users can watch the agent work.

## Get the live URL

Create a session first to get the `liveUrl`, then run a task in that session.

<CodeGroup>
```python Python
from browser_use_sdk import AsyncBrowserUse

client = AsyncBrowserUse()

session = await client.sessions.create()
print(session.live_url)  # embed this in an iframe

result = await client.run(
    "Search for flights from NYC to London",
    session_id=str(session.id),
)
```
```typescript TypeScript
import { BrowserUse } from "browser-use-sdk";

const client = new BrowserUse();

const session = await client.sessions.create();
console.log(session.liveUrl); // embed this in an iframe

const result = await client.run(
  "Search for flights from NYC to London",
  { sessionId: session.id },
);
```
</CodeGroup>

## Embed in HTML

```html
<iframe
  src="{live_url}"
  width="1280"
  height="720"
  allow="autoplay"
  style="border: none; border-radius: 8px;"
></iframe>
```

The live view updates in real time. No polling or WebSocket setup needed on your end.

## Public share links

Create a shareable public link for recording or sharing.

<CodeGroup>
```python Python
share = await client.sessions.create_share(session.id)
print(share.url)  # shareable URL anyone can view
```
```typescript TypeScript
const share = await client.sessions.createShare(session.id);
console.log(share.url);
```
</CodeGroup>
