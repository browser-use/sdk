This tutorial builds a therapist finder that searches [Rula](https://www.rula.com) — an insurance-backed online therapy marketplace — and returns structured results. We use [deterministic rerun](/cloud/agent/cache-script) so the first search uses the full agent, and subsequent searches with different criteria execute instantly at $0 LLM cost.

## What you'll build

A script that:
1. Searches Rula for therapists matching an insurance plan and specialty
2. Returns structured JSON with therapist name, specialties, and availability
3. Caches the search flow so follow-up searches are instant

## 1. Define the output schema

<CodeGroup>
```python Python
class Therapist(BaseModel):
    name: str
    specialties: list[str]
    insurance_accepted: list[str]
    next_available: str | None = None

class TherapistResults(BaseModel):
    therapists: list[Therapist]
    search_criteria: str
```
```typescript TypeScript
const TherapistResults = z.object({
  therapists: z.array(z.object({
    name: z.string(),
    specialties: z.array(z.string()),
    insuranceAccepted: z.array(z.string()),
    nextAvailable: z.string().nullable(),
  })),
  searchCriteria: z.string(),
});
```
</CodeGroup>

## 3. Search for therapists

Use `{{brackets}}` to mark the parameters that change between searches. The first call runs the full agent; subsequent calls with the same template are cached.

<CodeGroup>
```python Python
result = await client.run(
    "Go to rula.com and search for therapists who accept {{Aetna}} insurance "
    "and specialize in {{anxiety}}. Return the first 5 results as JSON.",
    workspace_id=str(workspace.id),
    schema=TherapistResults,
)

for t in result.output.therapists:
    print(f"{t.name} — {', '.join(t.specialties)}")
    print(f"  Insurance: {', '.join(t.insurance_accepted)}")
    print(f"  Next available: {t.next_available}")
    print()
```
```typescript TypeScript
const result = await client.run(
  "Go to rula.com and search for therapists who accept {{Aetna}} insurance " +
  "and specialize in {{anxiety}}. Return the first 5 results as JSON.",
  { workspaceId: workspace.id, schema: TherapistResults },
);

for (const t of result.output.therapists) {
  console.log(`${t.name} — ${t.specialties.join(", ")}`);
  console.log(`  Insurance: ${t.insuranceAccepted.join(", ")}`);
  console.log(`  Next available: ${t.nextAvailable}`);
}
```
</CodeGroup>

## 4. Run cached searches with different criteria

Now loop over different insurance plans and specialties. Each call reuses the cached script — $0 LLM cost, ~5 seconds.

<CodeGroup>
```python Python
searches = [
    ("UnitedHealthcare", "depression"),
    ("Blue Cross Blue Shield", "PTSD"),
    ("Cigna", "couples therapy"),
]

for insurance, specialty in searches:
    result = await client.run(
        f"Go to rula.com and search for therapists who accept {{{{{insurance}}}}} insurance "
        f"and specialize in {{{{{specialty}}}}}. Return the first 5 results as JSON.",
        workspace_id=str(workspace.id),
        schema=TherapistResults,
    )
    print(f"\n--- {insurance} / {specialty} ---")
    print(f"Found {len(result.output.therapists)} therapists")
    print(f"LLM cost: ${result.llm_cost_usd}")
```
```typescript TypeScript
const searches = [
  ["UnitedHealthcare", "depression"],
  ["Blue Cross Blue Shield", "PTSD"],
  ["Cigna", "couples therapy"],
];

for (const [insurance, specialty] of searches) {
  const result = await client.run(
    `Go to rula.com and search for therapists who accept {{${insurance}}} insurance ` +
    `and specialize in {{${specialty}}}. Return the first 5 results as JSON.`,
    { workspaceId: workspace.id, schema: TherapistResults },
  );
  console.log(`\n--- ${insurance} / ${specialty} ---`);
  console.log(`Found ${result.output.therapists.length} therapists`);
}
```
</CodeGroup>

## 5. Stream results in real time

For a better UX, stream the agent's progress as it navigates Rula:

<CodeGroup>
```python Python
stream = client.stream(
    "Go to rula.com and search for therapists who accept {{Aetna}} insurance "
    "and specialize in {{anxiety}}. Return the first 5 results as JSON.",
    workspace_id=str(workspace.id),
    schema=TherapistResults,
)

async for msg in stream:
    if msg.summary:
        print(f"[{msg.type}] {msg.summary}")

print(f"\nResult: {stream.result.output}")
```
```typescript TypeScript
const run = client.run(
  "Go to rula.com and search for therapists who accept {{Aetna}} insurance " +
  "and specialize in {{anxiety}}. Return the first 5 results as JSON.",
  { workspaceId: workspace.id, schema: TherapistResults },
);

for await (const msg of run) {
  if (msg.summary) console.log(`[${msg.type}] ${msg.summary}`);
}

console.log(`\nResult:`, run.result.output);
```
</CodeGroup>

## Summary

| Step | What happens | Cost |
|------|-------------|------|
| First search | Full agent navigates Rula, caches the flow | ~$0.10 |
| Subsequent searches | Cached script with new parameters | **$0 LLM** |
| Auto-heal (if site changes) | Agent regenerates the script automatically | ~$0.10 |

<Info>
If Rula changes their site layout, [auto-healing](/cloud/agent/cache-script#auto-healing) automatically detects the failure and regenerates the cached script — no manual intervention needed.
</Info>
