# Browser Use Cloud SDK

## Install
- Python: `pip install browser-use-sdk`
- TypeScript: `npm install browser-use-sdk`

## Setup
Set `BROWSER_USE_API_KEY` env var, or pass `api_key`/`apiKey` to the constructor.
Get a key at https://cloud.browser-use.com/settings?tab=api-keys&new=1

---

## Python SDK

```python
from browser_use_sdk import AsyncBrowserUse
from pydantic import BaseModel

client = AsyncBrowserUse()

# Run a task
result = await client.run("Find the top HN post")
print(result.output)   # str
print(result.id)       # session UUID
print(result.status)   # BuAgentSessionStatus (e.g. idle, stopped)

# Structured output
class Product(BaseModel):
    name: str
    price: float

result = await client.run("Get product info from amazon.com/dp/...", output_schema=Product)
print(result.output)  # Product(name=..., price=...)
```

### Constructor
```python
# Async client (recommended)
client = AsyncBrowserUse(api_key="...", base_url="...", timeout=30.0)

# Sync client
from browser_use_sdk import BrowserUse
client = BrowserUse(api_key="...", base_url="...", timeout=30.0)
```
- `api_key: str` ‚Äî default: `BROWSER_USE_API_KEY` env var
- `timeout: float` ‚Äî HTTP request timeout in seconds (default: `30.0`). Not the polling timeout.

### run() parameters
All optional keyword arguments:
- `model: str` ‚Äî `"bu-mini"` (default) or `"bu-max"` (more capable)
- `output_schema: type[BaseModel]` ‚Äî Pydantic model for structured output (alias: `schema`)
- `session_id: str` ‚Äî reuse an existing session
- `keep_alive: bool` ‚Äî keep session idle after task for follow-ups (default: `False`)
- `max_cost_usd: float` ‚Äî cost cap in USD; agent stops if exceeded
- `profile_id: str` ‚Äî persistent browser profile (cookies, localStorage)
- `proxy_country_code: str` ‚Äî residential proxy country (e.g. `"us"`, `"de"`)
- `workspace_id: str` ‚Äî attach a workspace for file persistence
- `enable_recording: bool` ‚Äî record session as MP4 (default: `False`)

`run()` returns an `AsyncSessionRun` (async) or `SessionResult` (sync):
- **AsyncSessionRun**: awaitable. After `await`, gives a `SessionResult`.
- Polling defaults: interval **2 seconds**, timeout **300 seconds** (5 min). Raises `TimeoutError` if exceeded.
- Terminal statuses: `idle`, `stopped`, `timed_out`, `error`.

### SessionResult fields
- `output` ‚Äî typed output (`str` or Pydantic model)
- `id` ‚Äî session UUID
- `status` ‚Äî `created`, `idle`, `running`, `stopped`, `timed_out`, `error`
- `model` ‚Äî `bu-mini` or `bu-max`
- `live_url` ‚Äî real-time browser monitoring URL (available after browser starts, poll `sessions.get()`)
- `recording_urls` ‚Äî list of MP4 download URLs (after task completes, if `enable_recording=True`)
- `total_input_tokens`, `total_output_tokens` ‚Äî token usage
- `llm_cost_usd`, `proxy_cost_usd`, `total_cost_usd` ‚Äî cost breakdown
- `created_at`, `updated_at` ‚Äî timestamps

### Sessions
```python
# Follow-up tasks in the same browser
session = await client.sessions.create()
result1 = await client.run("Log into example.com", session_id=str(session.id), keep_alive=True)
result2 = await client.run("Now click settings", session_id=str(session.id))
await client.sessions.stop(str(session.id))

# With profile (persistent login state)
session = await client.sessions.create(profile_id="your-profile-uuid")

# Live messages (poll for agent reasoning + tool calls)
msgs = await client.sessions.messages(str(session.id), limit=100)
for m in msgs.messages:
    print(f"[{m.role}] {m.data[:200]}")
```

### Workspaces (file handling)
```python
# Create workspace and run task that saves files
workspace = await client.workspaces.create(name="my-workspace")
result = await client.run("Go to HN and save top 3 posts as posts.json", workspace_id=str(workspace.id))

# Upload a file to workspace
upload = await client.workspaces.upload_files(str(workspace.id),
    files=[{"name": "data.csv", "contentType": "text/csv", "size": 100}])
# PUT file to upload.files[0].upload_url with matching Content-Type

# Download files from workspace
files = await client.workspaces.files(str(workspace.id), include_urls=True)
for f in files.files:
    print(f.path, f.url)  # presigned download URL
```

### Profiles
```python
profile = await client.profiles.create(name="user-id-1")
profiles = await client.profiles.list(query="user-id-1")  # search by name
await client.profiles.delete(str(profile.id))
```

### Browsers (direct CDP access)
```python
browser = await client.browsers.create(proxy_country_code="us")
print(browser.cdp_url)   # connect with Playwright/Puppeteer/Selenium
print(browser.live_url)
await client.browsers.stop(str(browser.id))

# Or skip the SDK: wss://connect.browser-use.com?apiKey=KEY&proxyCountryCode=us
```

### Error handling
```python
from browser_use_sdk import AsyncBrowserUse, BrowserUseError

try:
    result = await client.run("Do something")
except TimeoutError:
    print("SDK polling timed out (5 min default)")
except BrowserUseError as e:
    print(f"API error: {e}")
```

## TypeScript SDK

```typescript
import { BrowserUse } from "browser-use-sdk";
import { z } from "zod";

const client = new BrowserUse();

const result = await client.run("Find the top HN post");
console.log(result.output);

// Structured output (Zod)
const Product = z.object({ name: z.string(), price: z.number() });
const typed = await client.run("Get product info", { schema: Product });

// Sessions + follow-up tasks
const session = await client.sessions.create();
await client.run("Log in", { sessionId: session.id, keepAlive: true });
await client.run("Click settings", { sessionId: session.id });
await client.sessions.stop(session.id);

// Workspaces
const workspace = await client.workspaces.create({ name: "my-workspace" });
const res = await client.run("Save top 3 HN posts as posts.json", { workspaceId: workspace.id });
const files = await client.workspaces.files(workspace.id, { includeUrls: true });

// Browsers (direct CDP)
const browser = await client.browsers.create({ proxyCountryCode: "us" });
console.log(browser.cdpUrl);
await client.browsers.stop(browser.id);
```

### run() options (second argument)
- `model` ‚Äî `"bu-mini"` (default) or `"bu-max"`
- `schema` ‚Äî Zod schema for structured output
- `sessionId` ‚Äî reuse an existing session
- `keepAlive` ‚Äî keep session alive after task (default: `false`)
- `maxCostUsd` ‚Äî cost cap in USD
- `profileId` ‚Äî persistent browser profile UUID
- `proxyCountryCode` ‚Äî residential proxy country code
- `workspaceId` ‚Äî attach workspace for file persistence
- `enableRecording` ‚Äî record session as MP4
- `timeout` ‚Äî max polling time in ms (default: `300_000`)
- `interval` ‚Äî polling interval in ms (default: `2_000`)

## Key concepts
- **Task**: text prompt ‚Üí agent browses ‚Üí returns output
- **Session**: stateful browser sandbox. Auto-created by default, or create manually for follow-up tasks
- **Profile**: persistent browser state. Login once, reuse across sessions. Search by name with `profiles.list(query="...")`
- **Profile Sync**: upload local cookies to cloud: `curl -fsSL https://browser-use.com/profile.sh | sh`
- **Workspace**: persistent file storage across sessions. Upload input files, download results
- **Proxies**: US proxy on by default. Set `proxy_country_code` for other countries. 195+ countries. CAPTCHAs handled automatically
- **Stealth**: always on. Anti-detect browser, CAPTCHA solving, ad blocking, Cloudflare bypass
- **Models**: `bu-mini` (default, faster/cheaper) and `bu-max` (more capable)
- **Cost control**: set `max_cost_usd` to cap spending. Check `total_cost_usd` on the result
- **keep_alive**: if `true`, session stays idle after task for follow-ups
- **Live URL**: poll `sessions.get()` to get `live_url` once browser starts. Embed in iframe, customize with `?theme=light` or `?ui=false`
- **Recording**: set `enable_recording=True` to get MP4 download URLs after task completes
- **Integrations**: 1000+ integrations (Gmail, Slack, Calendar, etc.). Connect at cloud.browser-use.com/integrations
- **1Password**: auto-fill passwords and 2FA codes with `op_vault_id`

For v2 legacy SDK docs, see: https://docs.browser-use.com/cloud/legacy/agent
