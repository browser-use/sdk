See your workspaces at [cloud.browser-use.com/settings](https://cloud.browser-use.com/settings?tab=workspaces).

## Create and use a workspace

<CodeGroup>
```python Python
from browser_use_sdk import AsyncBrowserUse

client = AsyncBrowserUse()
workspace = await client.workspaces.create(name="my-workspace")

result = await client.run(
    'Go to Hacker News and save the first three posts as posts.json',
    workspace_id=str(workspace.id),
)
print(result.output)
```
```typescript TypeScript
import { BrowserUse } from "browser-use-sdk";

const client = new BrowserUse();
const workspace = await client.workspaces.create({ name: "my-workspace" });

const result = await client.run(
  'Go to Hacker News and save the first three posts as posts.json',
  { workspaceId: workspace.id },
);
console.log(result.output);
```
</CodeGroup>

## Upload a file

Upload a file to a session, then tell the agent to use it.

<CodeGroup>
```python Python
import httpx
from browser_use_sdk import AsyncBrowserUse

client = AsyncBrowserUse()
session = await client.sessions.create()

upload = await client.sessions.upload_files(
    str(session.id),
    files=[{"name": "data.csv", "content_type": "text/csv"}],
)

with open("data.csv", "rb") as f:
    async with httpx.AsyncClient() as http:
        await http.put(
            upload.files[0].upload_url,
            content=f.read(),
            headers={"Content-Type": "text/csv"},
        )

result = await client.run(
    "Analyze the uploaded data.csv and summarize the results",
    session_id=str(session.id),
)
```
```typescript TypeScript
import { BrowserUse } from "browser-use-sdk";
import { readFileSync } from "fs";

const client = new BrowserUse();
const session = await client.sessions.create();

const upload = await client.sessions.uploadFiles(session.id, {
  files: [{ name: "data.csv", contentType: "text/csv" }],
});

await fetch(upload.files[0].uploadUrl, {
  method: "PUT",
  headers: { "Content-Type": "text/csv" },
  body: readFileSync("data.csv"),
});

const result = await client.run(
  "Analyze the uploaded data.csv and summarize the results",
  { sessionId: session.id },
);
```
</CodeGroup>

## Download files

List files in a workspace and download them via presigned URLs.

<CodeGroup>
```python Python
import httpx

files = await client.workspaces.files(str(workspace.id), include_urls=True)
for f in files.files:
    print(f.path, f.size)

    # Download
    async with httpx.AsyncClient() as http:
        resp = await http.get(f.url)
        with open(f.path, "wb") as out:
            out.write(resp.content)
```
```typescript TypeScript
const files = await client.workspaces.files(workspace.id, { includeUrls: true });
for (const f of files.files) {
  console.log(f.path, f.size);

  // Download
  const resp = await fetch(f.url);
  const data = await resp.text();
  console.log(data);
}
```
</CodeGroup>

## Manage workspaces

<CodeGroup>
```python Python
workspace = await client.workspaces.get(workspace_id)
updated = await client.workspaces.update(workspace_id, name="renamed")
workspaces = await client.workspaces.list()
await client.workspaces.delete(workspace_id)
```
```typescript TypeScript
const workspace = await client.workspaces.get(workspaceId);
const updated = await client.workspaces.update(workspaceId, { name: "renamed" });
const workspaces = await client.workspaces.list();
await client.workspaces.delete(workspaceId);
```
</CodeGroup>

<Warning>
  Deleting a workspace permanently removes all its files. This cannot be undone.
</Warning>
