## client.run()

| Parameter | Python | TypeScript | Description |
|-----------|--------|------------|-------------|
| `task` | `str` | `string` | What you want the agent to do. |
| `model` | `str` | `string` | `"bu-mini"` (default, faster) or `"bu-max"` (more capable). |
| `output_schema` | `type[BaseModel]` | `schema: ZodSchema` | Return structured data matching this schema. |
| `session_id` | `str` | `sessionId: string` | Reuse an existing session for follow-up tasks. |
| `keep_alive` | `bool` | `keepAlive: boolean` | Keep session alive after task completes (default: `False`). |
| `workspace_id` | `str` | `workspaceId: string` | Attach a workspace for persistent file storage. |
| `profile_id` | `str` | `profileId: string` | Use a saved browser profile (cookies, login state). |
| `proxy_country_code` | `str` | `proxyCountryCode: string` | Residential proxy country code, e.g. `"us"`, `"de"` (default: `"us"`). |
| `max_cost_usd` | `float` | `maxCostUsd: number` | Cost cap in USD. Default: `$20`. Agent stops if exceeded. |
| `enable_recording` | `bool` | `enableRecording: boolean` | Record session as MP4 video (default: `False`). |

### Return value

`client.run()` polls until the task completes and returns a `SessionResult`:

| Field | Type | Description |
|-------|------|-------------|
| `id` | `str` | Session UUID. |
| `status` | `str` | See session statuses below. |
| `output` | `str` or structured type | The agent's result. |
| `model` | `str` | Model used (`bu-mini` or `bu-max`). |
| `title` | `str` | Auto-generated task title. |
| `is_task_successful` | `bool` | Whether the agent completed the task successfully. |
| `step_count` | `int` | Number of agent steps taken. |
| `last_step_summary` | `str` | Summary of the last step. |
| `live_url` | `str` | Real-time browser view URL. Null until browser starts — poll `sessions.get()`. |
| `recording_urls` | `list[str]` | MP4 download URLs (if `enable_recording=True`). |
| `screenshot_url` | `str` | Final page screenshot (presigned URL). |
| `total_cost_usd` | `str` | Total cost for this task. |
| `llm_cost_usd` | `str` | LLM token cost. |
| `proxy_cost_usd` | `str` | Proxy data transfer cost. |
| `browser_cost_usd` | `str` | Browser session cost. |
| `proxy_used_mb` | `str` | Proxy data transfer in MB. |
| `total_input_tokens` | `int` | Input tokens used. |
| `total_output_tokens` | `int` | Output tokens used. |
| `profile_id` | `str` | Profile used (if set). |
| `workspace_id` | `str` | Workspace used (if set). |
| `proxy_country_code` | `str` | Proxy country used. |
| `max_cost_usd` | `str` | Cost cap that was set. Default: `$20`. |
| `agentmail_email` | `str` | Unique email address for this session (e.g. `name@mail.bu.app`). |
| `created_at` | `str` | Session creation timestamp. |
| `updated_at` | `str` | Last update timestamp. |

### Session statuses

| Status | Meaning |
|--------|---------|
| `created` | Session created, no task running yet. |
| `running` | Agent is actively working on a task. |
| `idle` | Task completed, session alive (`keep_alive=True`). |
| `stopped` | Session ended. |
| `timed_out` | Session exceeded time limit. |
| `error` | Session encountered an error. |

## sessions.create()

Create a session manually for follow-up tasks or to get a `live_url` before running a task.

| Parameter | Python | TypeScript | Description |
|-----------|--------|------------|-------------|
| `task` | `str` | `task: string` | Optional. Omit to create an idle session. |
| `session_id` | `str` | `sessionId: string` | Dispatch to an existing idle session. |
| `model` | `str` | `model: string` | `"bu-mini"` (default) or `"bu-max"`. |
| `keep_alive` | `bool` | `keepAlive: boolean` | Keep alive after task (default: `False`). |
| `profile_id` | `str` | `profileId: string` | Browser profile to load. |
| `proxy_country_code` | `str` | `proxyCountryCode: string` | Proxy country. |
| `workspace_id` | `str` | `workspaceId: string` | Attach workspace. |
| `max_cost_usd` | `float` | `maxCostUsd: number` | Cost cap. |
| `output_schema` | `dict` | `outputSchema: object` | JSON Schema for structured output. |
| `enable_recording` | `bool` | `enableRecording: boolean` | Record as MP4. |

## sessions.messages()

Poll the agent's message history for building custom UIs.

| Parameter | Python | TypeScript | Description |
|-----------|--------|------------|-------------|
| `session_id` | `str` | `sessionId: string` | Session to get messages from. |
| `limit` | `int` | `limit: number` | Max messages to return (default: 50). |
| `after` | `str` | `after: string` | Cursor for forward pagination. |
| `before` | `str` | `before: string` | Cursor for backward pagination. |
